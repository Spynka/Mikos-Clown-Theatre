    О бренде

    Кодекс визуальной и смысловой идентичности театра клоунады «Микос»

    Кратко о бренде

    Экспериментальный театр, где клоунада становится языком глубокого диалога между сценой и зрителем.

    Цели бренда

    Создать узнаваемый культурный бренд с сильным визуальным кодом, привлечь новую интеллектуальную аудиторию.

    Задачи брендбука

    Единая система коммуникации: от афиш до соцсетей, сохранение характера «структурированного спектакля».

    Миссия

    Дарить зрителю опыт живого сопереживания через синтез клоунады, драмы и пластики. Смех, который остаётся в сердце.

    Ключевое сообщение

    «Искусство улыбаться всерьёз»

    Позиционирование

    Новаторский театр для думающей аудитории 20–45 лет, семьи, корпоративные клиенты.

    Задачи

    Рост узнаваемости на 40%, увеличение продаж билетов, формирование комьюнити вокруг бренда.

    Название / Слоган

    Театр клоунады «Микос» · «Игра, которая остаётся в сердце»

    Креативная концепция

    Контраст строгой геометрической типографики и живой клоунады. Красный акцент как «свет рампы».

    Визуальная стратегия

    Крупные эмоциональные портреты, высокий контраст, глубокий синий фон, золотые детали.

    Логотип

    Структура, модификации, охранное поле

    Основной
    (тёмный)

    Светлый вариант

    Красный акцент

    Структура

    Логотип состоит из знака (стилизованная маска) и текстовой части шрифтом Unbounded. Минимальный размер в печати — 25 мм.

    Охранное поле

    Равно высоте буквы «М» в слове «Микос». Запрещено размещать другие элементы ближе.

    Цветовые модификации

    Основной #231F20 на светлом фоне, светлый #F2F1F0 на тёмном, красный #F43139 — для специальных проектов.

    Ошибки: искажение пропорций, добавление теней, замена шрифта, цвет вне палитры.

    Фирменные цвета

    Палитра контрастов и театрального света

    Светлый фон #F2F1F0
    Тёмный #231F20
    Красный акцент #F43139
    Глубокий синий #2C4068
    Золотой #D4BF95

    Палитра

    Основная: #F2F1F0 + #231F20
    Акцентные: #F43139 (драйв), #2C4068 (интеллект), #D4BF95 (теплота сцены).

    Сочетания

    Тёмный текст на светлом, красный для кнопок/заголовков, синий для фоновых блоков. Запрещены сторонние оттенки.

    Типографика

    Unbounded (заголовки)

    Аа Бб Вв

    Для H1–H3, навигации, акцентных слов. Начертания: 600, 700, 800.

    Nunito Sans (текст)

    Аа Бб Вв

    Основной текст, описания, цитаты. Высокая читаемость. Начертания: 400, 600, 700.

    Правила использования

    • • Заголовки от 24px, межстрочный интервал 1.2
    • • Основной текст 16–18px, интерлиньяж 1.5
    • • Не более двух гарнитур в макете
    • • Избегать выравнивания по центру для длинных текстов

    Графические элементы

    Маска / знак

    Стилизованная клоунская маска — самостоятельный паттерн, используется на мерче и постерах.

    Линии кулис

    Вертикальные и горизонтальные полосы, напоминающие театральные кулисы — для динамики.

    Условия

    Только фирменные цвета. Не перекрывать логотип, соблюдать охранное поле.

    Фотостиль

    Эмоция, свет, театральный контраст

    Принципы фотостиля

    Естественное освещение, крупные портреты, высокий контраст (chiaroscuro), эмоциональная выразительность. Красный — единственный яркий акцент.

    Носители

    Применение бренда

    Внутренние носители

    Визитки, бланки, бейджи — логотип слева, охранное поле, шрифт Nunito Sans. Фирменные конверты с золотым акцентом.

    Внешние носители

    Афиши, билборды, соцсети — крупное фото, логотип вверху/снизу, красный акцент на датах. Мерч: минималистичный знак маски.

    Правила оформления

    Соблюдать охранное поле, использовать только палитру #F2F1F0, #231F20, #F43139, #2C4068, #D4BF95. Не нарушать читаемость.
